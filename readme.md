# Stephen G. Ware's CV

This repository contains
[Stephen G. Ware's curriculum vitae](https://sgware.github.io/cv/cv.pdf)
and the LaTeX source files used to create it. The LaTeX class is released under
the MIT License for anyone to modify or use.

The document can be compiled like this:
```
pdflatex cv.tex
bibtex cv
pdflatex cv.tex
```
